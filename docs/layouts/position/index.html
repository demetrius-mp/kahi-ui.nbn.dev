Position

Position is a layout primitive that can float itself and child content in specific regions of the containing Components.

Imports

<script>
    import {Position} from "@kahi-ui/framework";
</script>

Floated

You can set the Position to float its self (and its children) into a corner relative to its containing layout via the variation property.

Raised

You can set the Position to raise its self (and its children) in-place via the variation property.

API Reference

Position

Name Description Types
variation

Sets the Position to float on top of the containing layout, thus taking no physical space. Or raise it while taking space.
floated raised
placement

Sets the placement to top or bottom for variation="floated", and leftorrightforvariation="raised" properties.
bottom left right top
alignment_x

Sets the x-axis alignment of the variation="floated" property.
left right
