Breadcrumb

Breadcrumb is typically used to render a series of links with separators. With the each link left of the right-most a state in the end-user's navigation history. Or, a route hierarchy of the Web Application. e.g. directory paths

Imports

<script>
    import {Breadcrumb} from "@kahi-ui/framework";

    const {Container, Anchor, Item} = Breadcrumb;
</script>

Separator

You can alter the separator between items rendered via the separator property.

API Reference

Breadcrumb.Container

Name Description Types
separator

Alters the separator rendered between each breadcrumb item.
string

Breadcrumb.Item

Name Description Types
active

Alters the breadcrumb item to render bolder when active. Sets aria-current="page" on Breadcrumb.Anchor.
boolean

Breadcrumb.Anchor

Name Description Types
href

Changes the URL that Breadcrumb.Anchor navigates end-users to when clicked.
string
download

If linking to a downloadable file, this property will be used as the suggested file name.
string
rel

Sets how the Breadcrumb.Anchor relates to the current page, see MDN for more information.
string
target

Sets the target of the Breadcrumb.Anchor being navigated, see MDN for more information.
string
