Form

Form Components are helpers to supplement input Components that are meant for webforms like TextInput.

HelpText

Form.HelpText is typically used for displaying information about how to input information into a form properly.

Label

NOTE: This Component sets a Svelte Context for Form IDs / Names that child input Components consume.

Form.Label focuses the target input Component whenever its child content is clicked.

Control

NOTE: This Component sets a Svelte Context for Form IDs / Names that child input Components consume.

Form.Control is typically used to display form and input Components in a preformatted stack.

Group

NOTE: This Component is renderless, it does not render its own DOM nodes.

Form.Group let's you directly set the relevant form-related Svelte Contexts via its properties without using any preformatted shown above.
