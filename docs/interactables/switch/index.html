Switch

Switch is typically used to present to the end-user features, settings, and other toggles that they can turn on / off.

Imports

<script>
    import {Switch} from "@kahi-ui/framework";
</script>

States

You can control the state of the Switch via the active, disabled, and state properties.

Palette

You can change the color palette of the Switch via the palette property.

Size

You can change the size of the Switch via the size property.

API Reference

Switch

Name Description Types
palette

Alters the displayed color scheme.
auto inverse inherit accent dark light alert affirmative negative
size

Renders the Switch at a different sizes.
tiny small medium large huge
active

Renders the Switch with aria-pressed attribute, and styles the Switch as if it where being clicked.
boolean
disabled

Renders the Switch with disabled attribute, and styles the Switch partially transparent.
boolean
name

Sets the form name of the Switch.
string
state

Sets the checked state of the Switch.
boolean
value

Sets the value sent whenever the parent <form> is submitted.
string
