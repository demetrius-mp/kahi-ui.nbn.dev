Tab

Tab Components are typically used for turning what would be multi-page content, into a single page with a set of radio buttons to toggle between views.

Imports

<script>
    import {Tab} from "@kahi-ui/framework";

    const {
        Container,
        Anchor,
        Label,
        Group,
        Section,
    } = Tab;
</script>

Palette

You can change the color palette of the Tab radio buttons via the palette property.

Sizing

You can alter how large each Tab radio button should be via the sizing property.

Alignment

You can change the alignment of each Tab radio button via the alignment_x property.

Loading

NOTE: Use the in-Browser developer tools to see this functionality in action.

You can customize the slot loading behavior of individual Tab.Section Components to not render their children, via the loading property.

Anchor

You can substitute Tab.Label for Tab.Anchor to support per-page tabbed views.

API Reference

Tab.Container

logic_name

Description

Sets the form name attribute in a Svelte Context for child Tab.Label radio buttons to group as.
Types
string

logic_state

Description

Controls the currently active Tab.Label radio button via a Svelte Context.
Types
string

alignment_x

Description

Adjusts where the Tab.Anchor / Tab.Label radio buttons are, along the horizontal axis.
Types
center stretch left right

sizing

Description

Sets the size of each Tab.Anchor / Tab.Label radio buttons.
Types
tiny small medium large huge

Tab.Group

logic_id

Description

Sets the element id attribute in a Svelte Context for child Tab.Group / Tab.Label Components to group as.
Types
string

Tab.Anchor

current

Description

Changes the aria-current attribute, telling screen readers how to handle the link.
Types
date location page step time true

href

Description

Changes the URL that Tab.Anchor navigates end-users to when clicked.
Types
string

rel

Description

Sets how the Tab.Anchor relates to the current page, see MDN for more information.
Types
string

target

Description

Sets the target of the Tab.Anchor being navigated, see MDN for more information.
Types
string

palette

Description

Alters the displayed color scheme.
Types
auto inverse inherit accent dark light alert affirmative negative

Tab.Label

palette

Description

Alters the displayed color scheme.
Types
auto inverse inherit accent dark light alert affirmative negative

Tab.Section

loading

Description

When the property is set and the Tab.Section view is not active, the child content is not rendered to DOM.
Types
lazy

Tab.Anchor

click

Description

Fires whenever the Tab.Anchor is clicked.
Types
MouseEvent

Tab.Label

click

Description

Fires whenever the Tab.Label is clicked.
Types
MouseEvent
