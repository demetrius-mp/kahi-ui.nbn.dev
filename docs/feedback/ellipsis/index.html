Ellipsis

Ellipsis is a text-based indicator used to tell the end-user that the Web Application is loading something in the background.

Imports

<script>
    import {Ellipsis} from "@kahi-ui/framework";
</script>

Character

You can customize the text character used for the animation via the character property.

API Reference

Ellipsis

Name Description Types
character

Changes the character being rendered for the animation.
string

Ellipsis

Name Description Types
default

Default unnamed slot.
{}

Ellipsis

Name Description Types
--ellipsis-character

Controls the default character used, when no developer-supplied character is present.
<string>
--ellipsis-content-first-animation-delay

Controls how long of a delay until the first Ellipsis character is faded out in sequence.
<time>
--ellipsis-content-second-animation-delay

Controls how long of a delay until the second Ellipsis character is faded out in sequence.
<time>
--ellipsis-pulse-end-opacity

Controls how invisible a Ellipsis character should turn at the end of the "pulse" animation.
<alpha-value>
--ellipsis-pulse-start-opacity

Controls how visible a Ellipsis character should turn at the start of the "pulse" animation.
<alpha-value>
