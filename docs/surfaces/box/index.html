Box

Box is a surface primitive for encapsulating section content in a block of color.

Imports

<script>
    import {Box} from "@kahi-ui/framework";
</script>

Palette

You can change the color palette of the Box via the palette property.

Elevation

You can set how "high" your Box will appear to be over top the page via the elevation property.

Shape

You can modify the shape of the Box via the shape property.

API Reference

Box

Name Description Types
elevation

Alters how "high" the Box appears to be off the page.
none (DEFAULT) lowest low medium high highest
palette

Alters the displayed color scheme.
auto inverse inherit accent dark light alert affirmative negative
shape

Changes the shape of the Box.
none (DEFAULT) pill rounded

Box

Name Description Types
default

Default unnamed slot.
{}

Box

Name Description Types
--box-elevation

Controls the default box-shadow used.
<offset-x> <offset-y> <blur-radius> <spread-radius> <color>
--box-palette-background-normal

Controls the default RGB color value used for background.
<r,g,b>
--box-palette-foreground-normal

Controls the default RGB color value used for color.
<r,g,b>
--box-shape-radius

Controls the default border-radius used.
<length> <percentage>
