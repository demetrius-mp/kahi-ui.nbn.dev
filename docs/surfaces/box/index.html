Box

NOTE: New since v0.2.0.

Box is a surface primitive for encapsulating section content in a block of color.

<script>
    import {Box} from "@kahi-ui/framework";
</script>

<Box>This is a Box!</Box>

Imports

<script>
    import {Box} from "@kahi-ui/framework";
</script>

Elevation

NOTE: New since v0.3.5.

Shape

You can modify the shape of the Box via the shape property.

<script>
    import {
        Box,
        Mosaic,
        Text,
    } from "@kahi-ui/framework";
</script>

<Mosaic sizing="medium" spacing="large">
    <Box padding="huge">
        <Text is="strong">NONE / DEFAULT</Text>

        <Text>
            Lorem ipsum dolor sit amet, consectetur
            adipiscing elit. Proin et consectetur orci.
            Curabitur a egestas turpis, vitae convallis
            sapien. Sed pellentesque rutrum tellus, in

API Reference

Box

Name Description Types
elevation Alters how "high" the Box appears to be off the page.
none (DEFAULT) lowest low medium high highest
palette Alters the displayed color scheme.
auto inverse inherit accent dark light alert affirmative negative
shape Changes the shape of the Box.
none (DEFAULT) pill rounded

Box

Name Description Types
default Default unnamed slot.
{}

Box

Name Description Types
--box-elevation Controls the default box-shadow used.
<offset-x> <offset-y> <blur-radius> <spread-radius> <color>
--box-palette-background-normal Controls the default RGB color value used for background.
<r,g,b>
--box-palette-foreground-normal Controls the default RGB color value used for color.
<r,g,b>
--box-shape-radius Controls the default border-radius used.
<length> <percentage>
